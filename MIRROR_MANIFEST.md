# QScan Security Suite™ — Public Mirror Manifest

This repository is a **sanitized public mirror** of the QScan Security Suite™ source, part of the **QSS SynapseAI™** ecosystem.

## Included (Safe for Public Review)
- UI components (TypeScript / React)
- Documentation and diagrams
- Frontend stubs and integration examples
- Attestation templates (redacted)
- Non-sensitive configuration examples

## Excluded (Proprietary / Sensitive)
- Production backend source code
- API keys and environment variables
- Database schemas and live data connectors
- IBM Qiskit runtime secrets
- GPT / DeepSeek / Replit private integrations

## Purpose
This mirror supports:
- Public transparency
- Industry expert validation
- Visa 858 technical evidence
- External research collaborations
